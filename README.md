# rnGank

rnGank是又一个react native开发的gank.io的App。

目前只完成android端，感谢 [http://gank.io](http://gank.io) 提供 API。
### 功能

 - [x] 每日最新，随机推荐，技术，休闲界面
 - [x] 数据加载loading, 上拉下拉加载, 加载失败重试
 - [x] 实现自己的分配器( 主要用在 切换色调，语言，主题上 没用别的库
 - [x] 切换主色调, 语言, 主题(夜间模式
 - [x] 本地保存用户设置(主色调等等
 
### 运行

    yarn add or npm i
    react-native run-android
